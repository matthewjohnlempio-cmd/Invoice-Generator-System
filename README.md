# 📄 Invoice Generator System

A modern, full-stack invoice generator built with **Next.js**.  
Create professional invoices, manage clients, add services, and export clean printable previews instantly.

---

## ✨ Features

- 🧾 Dynamic invoice creation system  
- 👤 Client & business information management  
- 💼 Add services with hours and rates  
- 💰 Automatic subtotal & total calculations  
- 💳 Payment method support (GCash, Bank Transfer, PayPal)  
- 📝 Rich text notes editor (Quill-based)  
- 👀 Live invoice preview modal  
- 🖨️ Print / PDF-ready invoice layout  
- 🎨 Clean, modern, responsive UI  
- ⚡ Fast and lightweight Next.js app  

---

## 🛠️ Tech Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- Quill Rich Text Editor
